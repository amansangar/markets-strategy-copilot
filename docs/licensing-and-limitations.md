# Licensing And Limitations

This project is a local-first assessment application, not a commercial redistribution platform.

- Polygon, Finnhub, Alpha Vantage, FMP, NewsAPI, and FRED can have free-tier limits, redistribution restrictions, attribution expectations, and delayed data depending on account type.
- SEC EDGAR data is public but must be requested responsibly with a clear `SEC_USER_AGENT`.
- OpenAI is used only server-side for classification, concise explanations, clustering support, and report writing. It is never the direct source of BUY/SELL decisions.
- Alpaca support is paper portfolio sync only. The app does not implement real-money order execution.
- Performance metrics are generated from deterministic seeded data or user-selected historical data with stated assumptions. Do not present demo metrics as live trading performance.
- Macro regime and filing/event intelligence are contextual overlays. They are not forecasts and should not be interpreted as certainty.
