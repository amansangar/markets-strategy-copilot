# Two-Minute Demo Script

1. Open `/demo` and use it as the supervisor launchpad. Point out that it lists the safest route order, readiness checks, and core talking points.
2. Open the dashboard in demo mode and point out the live status banner, strongest buy / sell tiles, and the current market regime summary.
3. Use the left watchlist to select a symbol from a different asset class and show the chart, indicator overlays, and signal rationale.
4. Open the lower tabs and show:
   - cited news items
   - backtest gross vs net view
   - audit entries with reason codes and freshness
5. Switch to the scanner page and explain why the current symbol appeared.
6. Open the strategy tester, run a preset, and show walk-forward metrics, drawdown, and trade list.
7. Export an investment note PDF from the reports page and point to the saved file path in `artefacts/exports`.
8. Open settings, run the provider health check, and show how missing, degraded, or manual-check-needed providers surface honestly instead of fake freshness.
